# Calorigram Bot

Telegram бот для подсчета калорий с использованием ИИ для анализа фотографий еды.

## 🚀 Возможности

- 📸 **Анализ фотографий еды** с помощью ИИ
- 🎤 **Голосовой ввод** описания блюд
- 📝 **Текстовый ввод** описания блюд
- 📊 **Статистика питания** и отслеживание калорий
- 👤 **Персональные настройки** (возраст, рост, вес, активность)
- 💳 **Система подписок** через Telegram Stars
- 🔔 **Напоминания** о приемах пищи
- 👨‍💼 **Админ-панель** для управления пользователями

## 🛠 Технологии

- **Python 3.11+**
- **python-telegram-bot** - Telegram Bot API
- **SQLite** - база данных
- **Nebius AI** - анализ изображений
- **APScheduler** - планировщик задач
- **aiohttp** - асинхронные HTTP запросы

## 📦 Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/rilya888/calorigram_1.git
cd calorigram_1
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` с переменными окружения:
```env
BOT_TOKEN=your_telegram_bot_token
NEBUIS_API_KEY=your_nebius_api_key
BASE_URL=https://api.studio.nebius.ai/v1/
DATABASE_PATH=users.db
ADMIN_IDS=your_telegram_id
```

4. Запустите бота:
```bash
python main.py
```

## 🚀 Деплой на Railway

1. Подключите репозиторий к Railway
2. Установите переменные окружения в настройках Railway
3. Railway автоматически развернет бота

## 🔧 Конфигурация

### Переменные окружения

- `BOT_TOKEN` - токен Telegram бота
- `NEBUIS_API_KEY` - ключ API для Nebius AI
- `BASE_URL` - базовый URL API
- `DATABASE_PATH` - путь к файлу базы данных
- `ADMIN_IDS` - ID администраторов (через запятую)

### База данных

Бот автоматически создает и мигрирует базу данных SQLite с таблицами:
- `users` - пользователи
- `meals` - приемы пищи
- `calorie_checks` - проверки калорий
- `user_registration_history` - история регистраций

## 📊 Функции

### Для пользователей
- `/start` - запуск бота
- `/help` - справка
- `/profile` - профиль пользователя
- `/subscription` - управление подпиской
- `/reminders` - настройка напоминаний

### Для администраторов
- `/admin` - админ-панель
- `/resetcounters` - сброс счетчиков
