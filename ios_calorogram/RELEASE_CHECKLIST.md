# Чеклист подготовки к публикации в App Store

## 1. Настройка Apple Developer Account

- [ ] Создать Apple Developer Account (если еще нет)
- [ ] Оплатить годовую подписку ($99/год)
- [ ] Настроить команду разработчиков

## 2. Настройка App Store Connect

- [ ] Создать App ID в Apple Developer Portal
  - Bundle Identifier: `com.calorigram.app`
  - Capabilities: Sign in with Apple, Push Notifications (если нужно)
- [ ] Создать приложение в App Store Connect
- [ ] Заполнить информацию о приложении:
  - Название: Calorigram
  - Подзаголовок (если доступен)
  - Описание (на русском и английском)
  - Ключевые слова
  - Категории: Health & Fitness, Food & Drink
  - Контактная информация
  - Политика конфиденциальности URL

## 3. Подготовка скриншотов

- [ ] Скриншоты для iPhone (6.7", 6.5", 5.5")
- [ ] Скриншоты для iPad (12.9", 11")
- [ ] Промо-текст (до 170 символов)
- [ ] Иконка приложения (1024x1024)

## 4. Настройка проекта в Xcode

- [ ] Установить Bundle Identifier: `com.calorigram.app`
- [ ] Настроить Signing & Capabilities
- [ ] Выбрать Team в Signing
- [ ] Настроить Version и Build number
- [ ] Проверить Deployment Target (iOS 16.0+)
- [ ] Настроить App Icons
- [ ] Настроить Launch Screen

## 5. Тестирование

- [ ] Протестировать на реальных устройствах (iPhone и iPad)
- [ ] Протестировать на разных версиях iOS (16.0, 17.0, 18.0)
- [ ] Протестировать все функции:
  - [ ] Авторизация (Email, Phone, Apple)
  - [ ] Главный экран
  - [ ] Дневник
  - [ ] Статистика
  - [ ] Анализ блюд
  - [ ] Профиль
  - [ ] Напоминания
- [ ] Проверить работу в offline режиме
- [ ] Проверить обработку ошибок сети
- [ ] Проверить производительность

## 6. Правовые требования

- [ ] Политика конфиденциальности (обязательно для App Store)
- [ ] Условия использования (опционально)
- [ ] Подтверждение возраста (17+ если есть контент для взрослых)
- [ ] Экспорт соответствие (если используете шифрование)

## 7. Подготовка к сборке

- [ ] Очистить проект (Product → Clean Build Folder)
- [ ] Выбрать схему Release
- [ ] Archive приложение (Product → Archive)
- [ ] Проверить, что нет предупреждений компилятора
- [ ] Проверить, что нет утечек памяти

## 8. Загрузка в App Store Connect

- [ ] Загрузить архив через Xcode Organizer
- [ ] Дождаться обработки (обычно 10-30 минут)
- [ ] Проверить, что билд появился в App Store Connect
- [ ] Выбрать билд для отправки на ревью

## 9. Заполнение информации для ревью

- [ ] Контактная информация
- [ ] Демо-аккаунт (если требуется)
- [ ] Примечания для ревьюеров
- [ ] Информация о контенте (возрастные ограничения)

## 10. Отправка на ревью

- [ ] Проверить все поля еще раз
- [ ] Отправить на ревью
- [ ] Дождаться ответа (обычно 1-3 дня)

## 11. После одобрения

- [ ] Выбрать дату релиза (или выпустить сразу)
- [ ] Подготовить маркетинговые материалы
- [ ] Уведомить пользователей о релизе

## Частые проблемы

1. **Rejected: Missing Privacy Policy**
   - Решение: Добавить URL политики конфиденциальности в App Store Connect

2. **Rejected: Sign in with Apple**
   - Решение: Убедиться, что Apple Sign In работает корректно и является опциональным

3. **Rejected: Missing Demo Account**
   - Решение: Предоставить тестовый аккаунт в примечаниях для ревьюеров

4. **Rejected: App Crashes**
   - Решение: Протестировать на реальных устройствах перед отправкой
