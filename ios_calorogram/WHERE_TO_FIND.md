# Где найти настройки в Xcode - Визуальная инструкция

## 🎯 Signing & Capabilities

### Шаг за шагом:

1. **Откройте Xcode** (проект должен быть открыт)

2. **В левой панели** найдите:
   ```
   📁 Calorigram  ← Это СИНЯЯ иконка проекта (самый верхний элемент)
   ```
   ⚠️ НЕ путайте с желтой папкой "Calorigram" ниже!

3. **Кликните ОДИН РАЗ** на синюю иконку "Calorigram" (проект)

4. **В центральной панели** вы увидите:
   ```
   PROJECT
   ┌─────────────────────┐
   │ TARGETS             │
   │  • Calorigram       │ ← КЛИКНИТЕ СЮДА
   │  • CalorigramTests  │
   └─────────────────────┘
   ```

5. **Кликните ОДИН РАЗ** на "Calorigram" под TARGETS

6. **Вверху центральной панели** появятся вкладки:
   ```
   [General] [Signing & Capabilities] [Resource Tags] [Info] [Build Settings] [Build Phases] [Build Rules]
   ```

7. **Кликните на "Signing & Capabilities"**

8. **Вы увидите:**
   ```
   Signing
   ┌─────────────────────────────────────┐
   │ Automatically manage signing: [✓]   │
   │ Team: [None ▼]  ← КЛИКНИТЕ СЮДА    │
   │ Bundle Identifier: com.calorigram...│
   └─────────────────────────────────────┘
   
   Capabilities
   ┌─────────────────────────────────────┐
   │ ✅ Sign in with Apple               │
   └─────────────────────────────────────┘
   ```

## 📁 Проверка файлов в проекте

### Способ 1: В навигаторе (самый простой)

1. **В левой панели** найдите:
   ```
   📁 Calorigram  ← ЖЕЛТАЯ папка (не синяя иконка!)
   ```

2. **Кликните на стрелку** слева от папки, чтобы развернуть:
   ```
   ▶️ 📁 Calorigram
   ```
   Станет:
   ```
   ▼ 📁 Calorigram
      📁 App
      📁 Models
      📁 Services
      📁 ViewModels
      📁 Views
      📁 Utils
   ```

3. **Разверните каждую папку** и проверьте наличие файлов

### Способ 2: Через Build Phases

1. Проект → Target "Calorigram" → вкладка **"Build Phases"**

2. Разверните **"Compile Sources"**

3. Там должен быть список всех `.swift` файлов

### Способ 3: Поиск файла

1. Нажмите **⌘⇧F** (Command + Shift + F)
2. Введите имя файла, например: `HomeView`
3. Если файл найден - он в проекте

## ❓ Частые вопросы

**Q: Не вижу вкладку "Signing & Capabilities"**
- Убедитесь, что выбран **target "Calorigram"**, а не проект
- Вкладки находятся в **верхней части центральной панели**
- Попробуйте увеличить окно Xcode

**Q: Не вижу файлы в навигаторе**
- Убедитесь, что смотрите на **желтую папку**, а не на синюю иконку
- Разверните папку (кликните на стрелку)
- Если файлов нет, выполните: `xcodegen generate` в терминале

**Q: Где выбрать Team?**
- Проект → Target "Calorigram" → "Signing & Capabilities"
- В разделе "Signing" → "Team" → выпадающий список
